# AdventureWorks-SQL-Analysis-Project
This project is a complete SQL practice and analysis exercise performed on the AdventureWorksLT2022 database using SQL Server Management Studio (SSMS). It covers end-to-end SQL concepts used in real data analytics and database engineering work.

## Project Overview

# The project includes:
* Database Creation & Table Management
1.Creating a new database (sanjana_db)
2.Creating and dropping tables
3.Inserting, updating, and deleting records
4.Designing a custom table (table_marksheet)

* Data Exploration
1.Listing customers, products, addresses
2.Using aliases, ordering, filtering
3.Working with textual and numerical data

*SQL Operations Practiced
1.SELECT, WHERE, LIKE, IN, NOT IN
2.GROUP BY, ORDER BY, DISTINCT
3.COUNT, SUM, AVG
4.TOP, HAVING
5.CASE expressions
6.COALESCE() for NULL handling
7.CONCAT() for string manipulation

* Joins & Multi-Table Queries
1.Performed joins across key tables:
2.Customer
3.CustomerAddress
4.Address
5.SalesOrderHeader
6.SalesOrderDetail
7.Product
8.ProductModel
9.ProductDescription

Created a detailed combined dataset including:
1.Customer information
2.Address details
3.Product details
4.ORder summary (subtotal, tax, freight, totaldue)

* Analytical SQL
1.Monthly sales analysis using FORMAT()
2.Unit price & line total buckets using CASE
3.Product color distribution
4.High-value product identification with HAVING
5.Order volume categorization using IIF()

## Technologies Used

- SQL Server Management Studio (SSMS)
- T-SQL (Transact-SQL)
- AdventureWorksLT2022 Dataset
- Windows & SQL Express
